Screaming Strike 2!

Version : 2.00
作成者　: にゃんちゃん

このゲームは、近づいてくるいろんな敵をひたすら殴る、世界で一番ファニーなゲームです。
2012年に作ったネタ作品でしたが、なんと2018年に東京ゲームショウに出展されるとかいうすごいイベントが起こり、その流れで、真面目にリメイクしてみることにしました。

1:ゲームの特徴
三つのレーンのどこかから歩いてくるいろんなものを、ひたすら殴り倒していくゲームです。より正確に、より多くの敵をなぎ倒すノーマルモード、アイテムが登場し、さまざまな効果を発揮するアーケードモード、旧バージョンの雰囲気が好きな人のためのクラシックモードを収録しています。
操作はシンプルで、自分の出した成績をネットランキングに応募することもできます。
ネットランキングは世界中のユーザが参加でき、みんなで記録を競い合うことができます。

2:操作方法
今回、開発環境を変更した影響で、フォルダの中にけっこうファイルが散らばってしまいました。ゲームを起動するには、play.exeを実行してください。
ゲームを起動すると、メインメニューが出てきます。上下矢印とエンターキーで選択するか、メニュー項目の最後に読み上げられるショートカットキーを推して、直接項目を選ぶことができます。
メインメニューの一番上には、バージョン情報が表示されます。アップデートがある場合は、ここに通知されます。アップデートが検出されたときは、エンターキーを押すことで、最新版のインストーラーをダウンロードすることができます。ダウンロードをセットしたら、バックグラウンドで実行されますので、そのままゲームを遊んでいれば、その間にダウンロードしてくれます。
それぞれのモードを初めてプレイするときには、ヒントが表示されます。ヒントは、エンターキーで閉じることができます。
ゲームが始まると、左、中央、右のどこかから足音が聞こえてきます。左右の矢印で位置を合わせて、近づいてきたところをスペースで殴り倒してください。通常の状態では、敵がある程度接近していないと攻撃が当たらないので、気をつけてください。
最初は、自分のHPは3あります。敵に自分の位置まで接近されるとぶったたかれて、HPが1減ります。HPが0になると、倒れてゲームオーバーになってしまいます。
ゲームが終了すると、スコア送信をするかを聞かれますので、スコアをネットランキングに送信したかったらY、したくなかったらNを押してください。上下矢印とエンターキーで選択することもできます。
スコア送信では、送信するときにランキングに掲示される名前を決める必要があります。好きな名前を入力したら、エンターを押してください。

3:アーケードモード
アーケードモードでは、アイテムが時々出現します。アイテムが落ちてきたら、そのアイテムと同じ位置に立ってパンチを打ってください。アイテムも、パンチの命中範囲に入っていないと、取得することができません。
通常のパンチで叩いたアイテムは取得扱いとなり、効果が発動します。上矢印キーを押しながらアイテムを叩くと、アイテムの効果を発動させず、たたき壊すことができます。
アイテムには、よい効果をもたらすものと、悪い効果をもたらすものがあります。早く落ちてくるアイテムほど良いアイテムである確立が高く、ゆっくり落ちてくるアイテムほど悪いアイテムである確率が高いです(両方とも100%にはなりません)。
アイテムの効果は、持続系の効果であれば通常15秒間続きます。効果が切れる前に同じアイテムを複数取った場合は、それらの降下は重複して適用されるか、効果時間が延長されます。どちらになるかは、効果の種類によって違います。
アイテムの効果は以下の通りです。
よい効果:
megaton punch:離れた敵を攻撃できるようになります。連続して取った場合は、効果時間が延長されます。
Boost:パンチの速さが早くなります。連続して取った場合は、効果が重複します。
Penetration:命中範囲内の敵を貫通するようになり、この状態でパンチした敵からは、2倍のポイントを獲得することができます。連続して取った場合は、効果時間が延長されます。
Destruction:フィールドにいるすべての敵を殴り倒します。落下中のアイテムも一緒に壊れてしまいます。
悪い効果:
Shrink:パンチのリーチが短くなります。連臆して取った場合は、効果が重複します。
Slow down:パンチの速さが遅くなります。連続して取った場合は、効果が重複します。
Blurred:ものすごい勢いで位置が変化し、狙いがぶれます。連続して取った場合は、効果が重複します。

4:コレクション
このゲームには、たくさんの叫び声が収録されていて、その叫び声をコレクションできます。ゲーム中で10回殴った敵はコレクションに追加され、メインメニューのなかの「コレクション」画面から、いつでも叫ばせることができるようになります。
コレクション画面では、左右矢印キーを使ってコレクションを閲覧し、解放されているコレクションの上でエンターを推すと、そのキャラクターを叫ばせます。上下矢印を押すと、声の高さを変えられます。変えられる範囲は、ゲーム中にランダムで変化する範囲と共通です。また、シフトキーを押しながら左右矢印を押すことで、前/次の解放済みコレクションを検索できます。

5: データ削除
めいんめにゅーからアクセスできる「データ削除」メニューでは、ハイスコアと、コレクションのデータを個別で削除できます。実際にデータ削除を行うには、シフトキーを押しながらエンターキーを押してください。

6: オプション
各種設定を変更します。上下矢印で設定項目を移動し、左右矢印で値を変更します。アイテム・アナウンスの声は、今は1種類しかないので、左右矢印を押してもなにも起こりません。そのうち、だれかが声をくれたら追加するかもしれません。
